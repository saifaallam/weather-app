# Weather App

A simple weather app I built using HTML, CSS, and JavaScript. It lets you search for any city and shows the current weather conditions.

## What it does

- Search weather by city name
- Shows temperature, humidity, and wind speed
- Changes the background image based on the weather (sunny, rainy, cloudy, etc.)

## Technologies

- HTML / CSS / JavaScript
- [WeatherAPI](https://www.weatherapi.com/) for the weather data

## How to run it

1. Clone the repo
2. Get a free API key from [weatherapi.com](https://www.weatherapi.com/)
3. Open `script.js` and put your API key here :

```js
const API_KEY = "your_api_key";
```

4. Open `index.html` in your browser
